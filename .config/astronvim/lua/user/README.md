# nvim mappings cheatsheet

My custom bindings for nvim (using [AstroNvim](https://astronvim.github.io/).)

## Motion (normal and visual mode)

Shortcut | Action
---|---
 n | Cursor left
 i | Cursor right
 u | Cursor up
 e | Cursor down
   |
 N | Cursor to beginning of line
 I | Cursor to end of line
 U | Cursor up 5
 E | Cursor down 5
   |
 h | Back one page
 ' | Forward one page
